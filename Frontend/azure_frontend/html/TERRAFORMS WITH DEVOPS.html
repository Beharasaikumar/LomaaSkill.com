TERRAFORMS WITH DEVOPS

1. What is the primary purpose of Terraform in DevOps?

Correct Option: B) To automate the provisioning and management of infrastructure as code.

Solution and Explanation:
Terraform is an Infrastructure as Code (IaC) tool that allows you to define and manage infrastructure resources efficiently and consistently.

**Key Features of Terraform:**
1. Infrastructure as Code (IaC) for defining resources.
2. Supports multi-cloud and on-premises environments.
3. Declarative configuration language (HCL).
4. Automates deployment, scaling, and management of infrastructure.




2. Which of the following is a key feature of Azure DevOps?

Correct Option: B) Provides a comprehensive set of DevOps tools for CI/CD, version control, and more.

**Key Components of Azure DevOps:**
1. Azure Repos: Git repositories for version control.
2. Azure Pipelines: Continuous Integration and Continuous Deployment (CI/CD) services.
3. Azure Boards: Work tracking with Kanban boards and backlogs.
4. Azure Artifacts: Package management for sharing code.

16. What is the primary purpose of the `provider` block in a Terraform configuration for Azure?

Correct Option: A) To specify the cloud provider (Azure) and its configuration settings.

Solution and Explanation:
The `provider` block defines the cloud provider (Azure in this case) and includes details like subscription ID, tenant ID, and credentials.

**Example:**
```hcl provider "azurerm" { features {} subscription_id = "xxxx-xxxx-xxxx" client_id = "xxxx-xxxx-xxxx" client_secret = "xxxx" tenant_id = "xxxx" } ```
This allows Terraform to authenticate and interact with Azure resources.




17. Which Terraform block is used to define an Azure resource, such as a virtual network?

Correct Option: B) resource

Solution and Explanation:
The `resource` block is used to define the configuration for an Azure resource, such as a virtual network, storage account, or virtual machine.

**Example:**
```hcl resource "azurerm_virtual_network" "example" { name = "example-network" address_space = ["10.0.0.0/16"] location = "East US" resource_group_name = azurerm_resource_group.example.name } ```




18. How do you define a variable in Terraform for Azure resource configurations?

Variables in Terraform allow you to make your configurations dynamic and reusable.

**Example:**
