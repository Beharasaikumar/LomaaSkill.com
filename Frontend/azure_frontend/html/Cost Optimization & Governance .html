41. What is the definition of Role-Based Access Control (RBAC)?

Correct Option: D) A system for managing user access based on roles

Solution and Explanation:
RBAC is a system that restricts access to network resources based on a user's role within an organization. Each role is assigned permissions, and users inherit permissions based on the roles they are assigned to.

**Key Points**:
1. Users are assigned roles.
2. Roles are assigned permissions to perform certain actions.




42. What is the primary benefit of using RBAC in an organization?

Correct Option: C) It simplifies management by assigning permissions based on roles

Solution and Explanation:
RBAC simplifies access control by assigning users to predefined roles. Instead of assigning permissions to individual users, roles are assigned, and users inherit the permissions of those roles. This makes managing user access much easier, especially in large organizations.

**Steps in RBAC Implementation**:
1. Define roles in your system.
2. Assign users to these roles.
3. Grant permissions to roles, not individual users.




43. Which of the following is a key characteristic of RBAC?

Correct Option: A) Roles define a set of permissions

Solution and Explanation:
In RBAC, roles are the key components that define the permissions a user has. Each role in the system is linked to a set of permissions that allow users to perform specific actions or access certain resources.

**Explanation**:
1. A role is a collection of permissions.
2. Users are assigned roles, not individual permissions.




44. How does RBAC help enforce the principle of least privilege?

Correct Option: B) By assigning only necessary permissions to each role

Solution and Explanation:
RBAC enforces the principle of least privilege by ensuring that users only receive the minimum permissions necessary to perform their roles. This reduces the risk of unauthorized access to sensitive resources.

**Explanation**:
1. Roles are carefully defined to grant minimal access.
2. Users inherit only the permissions tied to their roles, limiting access to what they need.
