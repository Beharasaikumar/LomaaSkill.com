Feature Engineering

1. What is the main goal of feature engineering in data science?

Correct Option: A) To create better input features for machine learning models

Solution and Explanation:
Feature engineering involves transforming raw data into meaningful features that improve model performance.

Key Points:
1. Enhances the predictive power of models.
2. Reduces noise and improves accuracy.
3. Often includes encoding, scaling, and transformations.




2. Which of the following is an example of feature engineering?

Correct Option: B) One-hot encoding categorical variables

Solution and Explanation:
One-hot encoding transforms categorical variables into binary vectors so that they can be used in machine learning models.

Examples of feature engineering:
1. Normalization
2. Encoding categorical values
3. Creating interaction features




3. Why is feature scaling important in feature engineering?

Correct Option: C) It helps ensure features contribute equally to model training

Solution and Explanation:
Feature scaling ensures that all features are on the same scale, preventing bias in algorithms like KNN or SVM.

Key Benefits:
1. Improves convergence and performance.
2. Common techniques: Min-Max Scaling, Standardization.




4. Which technique is NOT typically part of feature engineering?

Correct Option: D) Hyperparameter tuning

Solution and Explanation:
Hyperparameter tuning is part of model optimization, not feature engineering. Feature engineering focuses on transforming and creating features.

Examples NOT part of feature engineering:
- Model selection
- Hyperparameter tuning




5. What is feature selection in the context of feature engineering?

Correct Option: A) Choosing the most relevant features for model training

Solution and Explanation:
Feature selection is the process of identifying and selecting features that are most useful for the model, improving performance and reducing overfitting.

Techniques include:
- Filter methods
- Wrapper methods
- Embedded methods




6. Which of the following is an example of a categorical feature?

Correct Option: B) Color of a car

Solution and Explanation:
Categorical features represent discrete categories or labels. "Color of a car" has values like Red, Blue, Green, etc., making it categorical.

Examples of categorical features:
- Gender
- Country
- Product type




7. What distinguishes numerical features from categorical ones?

Correct Option: A) Numerical features represent quantities and can be measured

Solution and Explanation:
Numerical features are quantitative and support arithmetic operations. They include integers and floating-point numbers.

Examples:
- Age
- Salary
- Temperature




8. Which of the following is a binary feature?

Correct Option: D) Whether a customer is active (Yes/No)

Solution and Explanation:
Binary features take only two possible values (like 0/1, Yes/No, True/False).

These are often encoded numerically to be used in models.




9. Which type of feature is suitable for mathematical operations?

Correct Option: C) Numerical

Solution and Explanation:

Types:
- Integer
- Float




10. Which of these features is ordinal in nature?

Correct Option: B) Customer satisfaction rating (Low, Medium, High)

Solution and Explanation:
Ordinal features have a meaningful order but no fixed numerical distance between them.

Examples:

- Satisfaction levels (Poor, Average, Good)




11. What is the most common reason for missing data in datasets?

Correct Option: C) Human error or non-response

Solution and Explanation:
Missing data often occurs due to user mistakes, skipped questions, or system errors during data collection. Handling these properly is crucial for model accuracy.




12. Which method of handling missing values involves replacing them with the mean or median?

Correct Option: A) Imputation

Solution and Explanation:
Imputation is a technique where missing values are replaced with statistical measures like mean, median, or mode to preserve data structure and distribution.




13. What is the drawback of simply removing rows with missing values?

Correct Option: B) It may result in loss of valuable data

Solution and Explanation:
While dropping rows is a quick fix, it can significantly reduce the dataset size, especially if many rows are affected, leading to biased or less accurate models.




14. Which method uses models to predict missing values?

Correct Option: C) Predictive imputation

Solution and Explanation:
Predictive imputation uses regression, KNN, or other models to estimate missing values based on other features in the dataset. It’s more accurate but computationally heavier.




15. Which of the following pandas functions can be used to drop missing values?

Correct Option: A) dropna()

Solution and Explanation:
The `dropna()` function in pandas is used to remove missing values (NaNs) from a DataFrame or Series. It can be customized to drop rows or columns.




16. What is the purpose of label encoding in data preprocessing?

Correct Option: B) To convert categorical data into numerical format

Solution and Explanation:
Label encoding transforms non-numerical labels (e.g., 'yes', 'no') into numeric values (e.g., 1, 0), which can be processed by machine learning algorithms.




17. Which library in Python provides the LabelEncoder class?

Correct Option: C) scikit-learn

Solution and Explanation:
`LabelEncoder` is part of the preprocessing module in the scikit-learn library, widely used for machine learning tasks in Python.




18. What is a potential drawback of label encoding?

Correct Option: D) It may imply ordinal relationships where none exist

Solution and Explanation:
Label encoding assigns numeric values to categories, which can mislead models into assuming a ranked relationship (e.g., 'apple'=0, 'banana'=1).




19. Which method is better than label encoding for nominal (non-ordinal) categories?

Correct Option: A) One-Hot Encoding

Solution and Explanation:
One-hot encoding avoids introducing order among categories and is more suitable for nominal data by creating binary columns for each category.




20. What does the LabelEncoder’s fit_transform() method do?

Correct Option: B) Fits the encoder and transforms the data

Solution and Explanation:
The `fit_transform()` method first learns the unique categories (fit) and then converts them to numeric values (transform) in one step.




21. What is One-Hot Encoding used for in data preprocessing?

Correct Option: A) To convert categorical variables into binary vectors

Solution and Explanation:
One-Hot Encoding creates binary (0 or 1) columns for each category, ensuring that no ordinal relationship is implied between them.




22. Which Python function is commonly used for One-Hot Encoding in pandas?

Correct Option: C) get_dummies()

Solution and Explanation:
The `get_dummies()` function in pandas creates a new column for each category, assigning 1 or 0 to indicate presence or absence.




23. What is a major drawback of One-Hot Encoding?

Correct Option: B) It can lead to high dimensionality

Solution and Explanation:
One-Hot Encoding creates a new column for every unique category. If the number of categories is large, this increases the feature space significantly.




24. How many binary columns will One-Hot Encoding create for a feature with 5 unique categories?

Correct Option: C) 5

Solution and Explanation:
One-Hot Encoding creates one column per unique category. Hence, 5 categories result in 5 binary columns unless "drop_first" is specified.




25. How can you reduce the dimensionality caused by One-Hot Encoding?

Correct Option: D) Use the 'drop_first=True' parameter

Solution and Explanation:
Setting `drop_first=True` in `get_dummies()` removes one column to avoid the dummy variable trap, reducing dimensionality slightly.




26. What is the primary purpose of Ordinal Encoding in data preprocessing?

Correct Option: B) To assign an integer to categories with order

Solution and Explanation:
Ordinal Encoding is used when categories have a meaningful order. It replaces each label with an integer value based on its rank.

28. What is a possible drawback of Ordinal Encoding?

Solution and Explanation:
If used on nominal data, ordinal encoding can wrongly imply a relationship or distance between categories.




29. Which Python library provides an OrdinalEncoder class?

Scikit-learn provides the `OrdinalEncoder` class as part of its preprocessing module to transform ordered categories into integers.




30. Which of the following data is suitable for Ordinal Encoding?

Education levels have a natural ordering, which makes them perfect candidates for ordinal encoding.




31. What is Binary Encoding used for in feature engineering?

Correct Option: C) To reduce dimensionality by encoding categories into binary digits

Solution and Explanation:
Binary Encoding first converts categories into numeric labels, then encodes those labels into binary code, reducing the number of columns created.




32. What is the first step in Binary Encoding?

Correct Option: A) Assign numerical values to each category

Solution and Explanation:
Binary Encoding begins by converting each category label into a numerical form, then encoding that number in binary format.




33. What is one main benefit of Binary Encoding compared to One-Hot Encoding?

Correct Option: D) It creates fewer columns

Solution and Explanation:
Binary Encoding is more memory efficient, especially for categorical features with many levels, as it generates fewer binary columns.




34. Binary Encoding is a combination of which two techniques?

Correct Option: B) Label Encoding and Binary Conversion

Solution and Explanation:
It combines label encoding with binary representation of integers, offering a balance between simplicity and dimensionality.




35. Which library is commonly used for Binary Encoding in Python?

Correct Option: C) category_encoders

Solution and Explanation:
The `category_encoders` library provides multiple encoders including BinaryEncoder, useful for efficient encoding of categorical data.




36. What is the key idea behind Frequency Encoding?

Correct Option: B) Replace each category with its frequency in the dataset

Solution and Explanation:
Frequency Encoding replaces each category in a feature with the number of times it appears in the dataset, which helps to preserve some information about the distribution.




37. Frequency Encoding is most useful when:

Correct Option: C) There are high-cardinality categorical variables

Solution and Explanation:
Frequency Encoding helps reduce dimensionality, which is especially useful when dealing with categorical variables that have many unique values.




38. What is a potential issue with Frequency Encoding?

Correct Option: A) It may mislead models into thinking frequencies have order

Solution and Explanation:
Frequencies are numeric and may trick the model into assuming a meaningful distance or hierarchy between them, which may not exist.




39. What is the concept of Target Encoding?

Correct Option: D) Replacing category with mean of target variable for that category

Solution and Explanation:
Target Encoding replaces each category with the average of the target variable for that category, making it a supervised encoding method.




40. What is one major risk of using Target Encoding?

Correct Option: C) It can lead to data leakage

Solution and Explanation:
Target Encoding uses the target variable to compute encodings. If applied before splitting the dataset, it may introduce data leakage and overfitting.




41. What is the main purpose of Hash Encoding?

Correct Option: B) Convert high-cardinality categorical variables into numeric vectors using hash functions

Solution and Explanation:
Hash Encoding uses a hash function to map categories to numeric values, which is especially useful for handling high-cardinality categorical variables.




42. What is a limitation of Hash Encoding?

Correct Option: C) Possibility of hash collisions

Solution and Explanation:
Hash Encoding may result in different categories being mapped to the same hash value, known as collisions, which may reduce model accuracy.




43. What is Leave-One-Out Encoding?

Correct Option: D) Replacing a category with the mean of the target excluding the current row

Solution and Explanation:
Leave-One-Out Encoding avoids data leakage by computing the mean of the target for each category excluding the row being encoded.




44. Why is Leave-One-Out Encoding preferred over Target Encoding?

Correct Option: B) It reduces data leakage

Solution and Explanation:
By excluding the current row from the calculation, Leave-One-Out Encoding prevents the model from indirectly accessing the target variable of the row being predicted.




45. What does Min-Max Scaling do?

Correct Option: A) Scales data to a fixed range like 0 to 1

Solution and Explanation:
Min-Max Scaling transforms features by scaling them to a specific range, usually [0, 1], which helps normalize input for many algorithms.




46. When should you avoid using Min-Max Scaling?

Correct Option: C) When dataset contains outliers

Solution and Explanation:
Min-Max Scaling is sensitive to outliers, which can compress the range of most data points and distort the scaled output.




47. What does Z-score Standardization do?

Correct Option: D) Transforms features to have mean 0 and standard deviation 1

Solution and Explanation:
Standardization scales data based on its distribution, ensuring each feature has a mean of 0 and a standard deviation of 1.




48. What is an advantage of Standardization over Min-Max Scaling?

Correct Option: A) It is less affected by outliers

Solution and Explanation:
Standardization is generally more robust in the presence of outliers compared to Min-Max Scaling, which can be skewed by extreme values.




49. What is the key advantage of using Robust Scaling in feature engineering?

Correct Option: B) It is less sensitive to outliers

Solution and Explanation:
Robust Scaling uses the median and interquartile range, making it effective when the dataset contains outliers.

Key Points:
1. Unlike Min-Max or Standard scaling, it doesn't get skewed by extreme values.
2. It centers the data using the median.
