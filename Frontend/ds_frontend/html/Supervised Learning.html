Supervised Learning

1. What is the main goal of supervised learning in data science?

Correct Option: A) To predict outputs based on labeled input data

Solution and Explanation:
Supervised learning uses labeled datasets to train algorithms to classify data or predict outcomes accurately.

Key Points:
1. Input and output data are both provided.
2. The model learns a mapping function from input to output.
3. Examples include regression and classification tasks.




2. Which of the following is an example of a supervised learning algorithm?

Correct Option: C) Linear Regression

Solution and Explanation:
Linear Regression is a popular supervised learning algorithm used to predict numerical values based on input features.

Examples of Supervised Algorithms:
1. Linear Regression (for regression tasks).
2. Decision Trees, SVM, KNN (for classification tasks).




3. In supervised learning, what type of data is required to train the model?

Correct Option: B) Labeled data

Solution and Explanation:
Labeled data includes both input features and their corresponding correct outputs, which are essential for supervised learning models to learn relationships.

Why it matters:
1. Provides feedback to the model.
2. Essential for training accurate models.




4. Which task is typically NOT solved using supervised learning?

Correct Option: D) Grouping customers without prior labels

Solution and Explanation:
Grouping without prior labels is a clustering task, which falls under unsupervised learning, not supervised learning.

Supervised vs Unsupervised:
- Supervised: Classification, Regression
- Unsupervised: Clustering, Association




5. What are the two main types of problems in supervised learning?

Correct Option: C) Classification and Regression

Solution and Explanation:
Supervised learning deals with two main types of problems:

Classification – Predicting categories (e.g., spam or not spam).
Regression – Predicting continuous values (e.g., price of a house).




6. Which of the following is a type of supervised learning that deals with predicting continuous values?

Correct Option: B) Regression

Solution and Explanation:
Regression is used in supervised learning to predict continuous numerical outcomes based on input variables.

Example: Predicting salary based on experience and education.




7. In classification problems, what is the nature of the output variable?

Correct Option: A) Categorical

Solution and Explanation:
Classification problems aim to categorize inputs into discrete classes or categories.

Example: Classifying emails into spam or not spam.




8. Which algorithm is commonly used for classification tasks?

Correct Option: C) Decision Trees

Solution and Explanation:
Decision Trees are widely used for classification due to their simplicity and interpretability.

Other classification algorithms include KNN, SVM, and Logistic Regression.




9. What type of supervised learning problem is best suited for predicting stock prices?

Correct Option: D) Regression

Solution and Explanation:
Stock prices are continuous numerical values, making regression the ideal supervised learning approach for prediction tasks.




10. Which of the following best describes the difference between classification and regression?

Correct Option: B) Classification predicts categories, regression predicts numeric values

Solution and Explanation:
Classification is used when the output is a discrete label, whereas regression is used for continuous outcomes.
Example: Classification → Spam/Not spam
Regression → Predicting temperature




11. What is the main purpose of data preprocessing in a machine learning pipeline?

Correct Option: C) To clean and prepare raw data for model training

Solution and Explanation:
Data preprocessing involves cleaning, transforming, and organizing data to improve the performance of machine learning models.




12. Which of the following is NOT a common data preprocessing step?

Correct Option: D) Model evaluation

Solution and Explanation:
Model evaluation is part of model assessment, not data preprocessing. Preprocessing includes handling missing data, scaling, and encoding.




13. Which method is used to handle missing values in a dataset?

Correct Option: B) Imputation

Solution and Explanation:
Imputation fills in missing values using statistical techniques such as mean, median, or mode substitution.




14. What does normalization do to the data?

Correct Option: C) Scales the data to a specific range

Solution and Explanation:
Normalization transforms data into a fixed range (like 0 to 1), helping models converge faster and improve performance.




15. One-hot encoding is typically used for:

Correct Option: A) Converting categorical data into binary vectors

Solution and Explanation:
One-hot encoding is used to represent categorical variables as binary vectors to be used in machine learning models.




16. What is feature engineering in machine learning?

Correct Option: B) The process of creating new input features from existing ones

Solution and Explanation:
Feature engineering involves creating new variables or transforming existing ones to improve model performance.




17. Which of the following is a feature extraction technique?

Correct Option: A) Principal Component Analysis (PCA)

Solution and Explanation:
PCA reduces the dimensionality of data by extracting important features (principal components) from it.




18. What is one advantage of using polynomial features in feature engineering?

Correct Option: C) It allows the model to capture non-linear relationships

Solution and Explanation:
Polynomial features help models detect curves or higher-order patterns that simple linear models can't capture.




19. What is the purpose of feature selection?

Correct Option: D) To reduce overfitting and improve model performance

Solution and Explanation:
Feature selection removes irrelevant or redundant features, improving generalization and reducing computational cost.




20. Which of the following can be considered a domain-specific feature engineering task?

Correct Option: A) Extracting month from a date field in a sales dataset

Solution and Explanation:
Domain-specific feature engineering uses knowledge of the dataset's context (e.g., sales, health) to create meaningful features.




21. What is the purpose of train-test split in machine learning?

Correct Option: B) To evaluate model performance on unseen data

Solution and Explanation:
Train-test split separates data into training and testing sets to assess how well the model generalizes to new data.




22. What is a common ratio used in train-test splitting?

Correct Option: C) 80% training, 20% testing

Solution and Explanation:
A typical split is 80% of the data for training and 20% for testing, though this may vary based on dataset size and context.




23. Why is it important that the test data is never seen by the model during training?

Correct Option: A) To fairly assess how the model performs on new, unseen data

Solution and Explanation:
Using unseen test data ensures a realistic evaluation of the model's generalization performance.




24. Which library function is commonly used to perform train-test split in Python?

Correct Option: D) train_test_split() from sklearn.model_selection

Solution and Explanation:
`train_test_split()` is a utility from Scikit-learn used to divide data into training and test sets.




25. What issue might occur if you don’t perform a proper train-test split?

Correct Option: C) The model may overfit and perform poorly on unseen data

Solution and Explanation:
Without a proper train-test split, the model might learn patterns from both sets, leading to high training accuracy but poor generalization.




26. What is the main goal of cross-validation in machine learning?

Correct Option: A) To evaluate model performance more reliably

Solution and Explanation:
Cross-validation helps assess a model's performance by averaging results over multiple splits of the data, reducing bias and variance.




27. In K-Fold Cross-Validation, what does 'K' represent?

Correct Option: B) The number of groups data is split into

Solution and Explanation:
In K-Fold, the dataset is divided into K parts (folds), and the model is trained and validated K times, each time with a different fold as validation.




28. What is a common value of K in K-Fold Cross-Validation?

Correct Option: C) 5 or 10

Solution and Explanation:
5-fold and 10-fold are widely used as they provide a good balance between bias and variance.




29. What is one downside of using cross-validation?

Correct Option: D) It increases computational cost

Solution and Explanation:
Since the model is trained multiple times, cross-validation requires more computation and time.




30. Which cross-validation strategy ensures each fold has the same class distribution as the whole dataset?

Correct Option: B) Stratified K-Fold

Solution and Explanation:
Stratified K-Fold keeps the class ratio balanced across each fold, which is especially important in imbalanced classification tasks.




31. Which of the following is NOT an evaluation metric for classification?

Correct Option: D) Mean Squared Error

Solution and Explanation:
Mean Squared Error (MSE) is used in regression problems, not classification. Classification metrics include accuracy, precision, recall, etc.




32. What does precision measure in a classification problem?

Correct Option: A) Correct positive predictions out of total predicted positives

Solution and Explanation:
Precision = True Positives / (True Positives + False Positives). It tells how many predicted positives are actually correct.




33. What does recall (or sensitivity) measure?

Correct Option: C) Correct positive predictions out of actual positives

Solution and Explanation:
Recall = True Positives / (True Positives + False Negatives). It measures the ability to detect all actual positives.




34. Which metric gives a balance between precision and recall?

Correct Option: B) F1 Score

Solution and Explanation:
F1 Score is the harmonic mean of precision and recall. It provides a single metric that balances both concerns.




35. What does the ROC curve represent?

Correct Option: A) Trade-off between true positive rate and false positive rate

Solution and Explanation:
The ROC (Receiver Operating Characteristic) curve plots TPR vs. FPR to show the diagnostic ability of a binary classifier at various thresholds.




36. Which of the following is a commonly used metric to evaluate regression models?

Correct Option: A) Mean Absolute Error

Solution and Explanation:
Mean Absolute Error (MAE) measures the average magnitude of errors in a set of predictions, without considering their direction.




37. What does Mean Squared Error (MSE) measure in regression?

Correct Option: B) The average of the squares of the errors

Solution and Explanation:
MSE calculates the average of the squared differences between actual and predicted values. Squaring penalizes larger errors more.




38. Which metric is the square root of Mean Squared Error?

Correct Option: D) Root Mean Squared Error

Solution and Explanation:
RMSE = √MSE. It interprets the error in the same unit as the target variable and penalizes large errors more heavily.




39. What does R² score (coefficient of determination) indicate?

Correct Option: C) Proportion of variance explained by the model

Solution and Explanation:
R² explains how much variability in the target variable can be explained by the model. 1 means perfect prediction, 0 means no explanatory power.




40. Which of the following regression metrics is most sensitive to outliers?

Correct Option: B) Mean Squared Error

Solution and Explanation:
MSE squares the errors, so larger errors contribute more to the total, making it sensitive to outliers compared to MAE.




41. What is the primary goal of Linear Regression?

Correct Option: A) To find the best-fitting line through the data

Solution and Explanation:
Linear Regression aims to model the relationship between independent and dependent variables by fitting a linear equation to the data.




42. What is the equation of a simple linear regression model?

Correct Option: C) y = β₀ + β₁x

Solution and Explanation:
Simple Linear Regression represents the relationship between one independent variable x and a dependent variable y using the formula y = β₀ + β₁x.




43. Which method is commonly used to find the best-fit line in Linear Regression?

Correct Option: B) Least Squares Method

Solution and Explanation:
The least squares method minimizes the sum of the squares of the differences between observed and predicted values.




44. What does the coefficient β₁ represent in the linear regression equation?

Correct Option: D) The slope of the regression line

Solution and Explanation:
β₁ represents the change in the dependent variable y for each one-unit change in the independent variable x.




45. Which assumption is NOT required for simple linear regression?

Correct Option: A) Multicollinearity

Solution and Explanation:
Multicollinearity is an issue in multiple regression, not simple linear regression which involves only one independent variable.




46. What type of problem is Logistic Regression used to solve?

Correct Option: C) Classification problems

Solution and Explanation:
Logistic Regression is used for binary or multiclass classification tasks, predicting probabilities of categories.




47. What is the output of a Logistic Regression model?

Correct Option: B) Probability between 0 and 1

Solution and Explanation:
Logistic Regression uses the sigmoid function to output probabilities between 0 and 1 which represent the likelihood of a class.




48. Which function is used in Logistic Regression to map values to probabilities?

Correct Option: C) Sigmoid function

Solution and Explanation:
The sigmoid function maps the linear combination of inputs to a value between 0 and 1, enabling classification.




49. What loss function is used in Logistic Regression?

Correct Option: A) Log Loss (Binary Cross-Entropy)

Solution and Explanation:
Logistic Regression minimizes the Log Loss, which measures the performance of a classification model with probability outputs.




50. Which of the following is a limitation of Logistic Regression?

Correct Option: D) It assumes linearity between independent variables and the log-odds

Solution and Explanation:
Logistic Regression assumes a linear relationship between the input variables and the log-odds of the output, which may not always hold true.




51. What is the main idea behind Decision Trees?

Correct Option: A) Splitting data based on feature values to make decisions

Solution and Explanation:
Decision Trees split the data based on the values of features to create branches leading to a decision outcome.




52. Which of the following is used to measure the quality of a split in Decision Trees?

Correct Option: B) Gini Impurity

Solution and Explanation:
Gini Impurity is commonly used in classification trees to measure the impurity of a dataset after a split.




53. What does a leaf node represent in a Decision Tree?

Correct Option: D) Final decision or classification

Solution and Explanation:
Leaf nodes represent the final predicted output after all splits have been made in the decision process.




54. Which of the following is a disadvantage of Decision Trees?

Correct Option: A) They are prone to overfitting

Solution and Explanation:
Decision Trees can overfit the training data, especially if not properly pruned or regularized.




55. What is pruning in the context of Decision Trees?

Correct Option: B) Removing branches that have little importance

Solution and Explanation:
Pruning is a technique used to reduce the size of a tree by removing sections that provide little power to classify instances.




56. What is a Random Forest in machine learning?

Correct Option: C) An ensemble of decision trees

Solution and Explanation:
Random Forest is an ensemble learning method that builds multiple decision trees and merges their results for better accuracy and stability.




57. What technique does Random Forest use to ensure diversity among the trees?

Correct Option: A) Bagging (Bootstrap Aggregation)

Solution and Explanation:
Random Forest uses Bagging, where multiple datasets are sampled with replacement, ensuring varied training data for each tree.




58. In Random Forest, what happens during prediction for classification?

Correct Option: D) Majority voting from all trees

Solution and Explanation:
In classification problems, each tree gives a prediction and the Random Forest takes the majority vote to decide the final output.




59. What is one of the advantages of Random Forest over a single Decision Tree?

Correct Option: B) It reduces overfitting

Solution and Explanation:
Combining multiple trees helps generalize better and reduces the overfitting problem common in single decision trees.




60. Which hyperparameter in Random Forest controls the number of trees in the model?

Correct Option: C) n_estimators

Solution and Explanation:
The `n_estimators` parameter defines how many decision trees will be created in the forest.




61. What is the main idea behind Gradient Boosting?

Correct Option: A) Combine weak learners to minimize errors

Solution and Explanation:
Gradient Boosting builds an ensemble of weak learners (usually decision trees) sequentially, each trying to correct the errors of the previous one.




62. How does Gradient Boosting update its model?

Correct Option: C) By fitting new models to residuals

Solution and Explanation:
Gradient Boosting trains each new model on the residual errors of the previous model to reduce the overall prediction error.




63. Which type of learners are typically used in Gradient Boosting?

Correct Option: B) Decision trees

Solution and Explanation:
Gradient Boosting usually uses shallow decision trees (also called stumps) as base learners to build the model.




64. What does the learning rate in Gradient Boosting control?

Correct Option: A) The contribution of each tree to the final model

Solution and Explanation:
The learning rate scales the impact of each additional model. A lower learning rate means slower learning but possibly better accuracy.




65. Which of the following is a common problem with Gradient Boosting?

Correct Option: C) Overfitting on noisy data

Solution and Explanation:
Gradient Boosting is powerful but sensitive to noise, and can overfit easily without careful tuning and regularization.




66. What does XGBoost stand for?

Correct Option: B) Extreme Gradient Boosting

Solution and Explanation:
XGBoost stands for Extreme Gradient Boosting. It's a scalable and efficient implementation of gradient boosting machines.




67. What makes XGBoost faster than traditional Gradient Boosting?

Correct Option: A) Parallel processing and regularization

Solution and Explanation:
XGBoost uses system optimization like parallel processing, cache awareness, and regularization techniques to speed up training and reduce overfitting.




68. Which regularization techniques are used in XGBoost?

Correct Option: D) L1 and L2 regularization

Solution and Explanation:
XGBoost incorporates both L1 (Lasso) and L2 (Ridge) regularization to reduce overfitting and control complexity of the trees.




69. What type of data structure does XGBoost use for efficient computation?

Correct Option: B) DMatrix

Solution and Explanation:
XGBoost uses a special data structure called DMatrix which is optimized for memory and computation speed.




70. What is the objective function in XGBoost composed of?

Correct Option: A) Loss function + Regularization term

Solution and Explanation:
The objective function in XGBoost is composed of a loss function (measuring training error) and a regularization term (to avoid overfitting).




71. What is LightGBM mainly designed for?

Correct Option: A) High performance and efficiency on large datasets

Solution and Explanation:
LightGBM is optimized for high speed and performance, particularly with large datasets and high-dimensional data.




72. Which technique does LightGBM use for tree growth?

Correct Option: B) Leaf-wise growth

Solution and Explanation:
LightGBM grows trees leaf-wise (best-first) instead of level-wise like traditional methods, which increases accuracy.




73. What data structure helps LightGBM handle large datasets efficiently?

Correct Option: D) Histogram-based algorithm

Solution and Explanation:
LightGBM uses a histogram-based algorithm for faster training and reduced memory usage.




74. LightGBM is sensitive to which kind of feature?

Correct Option: A) Categorical features

Solution and Explanation:
LightGBM supports categorical features natively, but improper encoding or treatment can affect performance.




75. What is one disadvantage of LightGBM?

Correct Option: C) May overfit on small datasets

Solution and Explanation:
LightGBM is very powerful but can easily overfit if used on small datasets or without proper parameter tuning.




76. What is Scikit-learn primarily used for in data science?

Correct Option: B) Machine learning modeling and evaluation

Solution and Explanation:
Scikit-learn is a Python library widely used for machine learning tasks including model building, training, and evaluation.




77. Which of the following tasks can be performed using Scikit-learn?

Correct Option: D) All of the above

Solution and Explanation:
Scikit-learn supports classification, regression, clustering, dimensionality reduction, model selection, and preprocessing.




78. Which method is used to split data into training and testing sets in Scikit-learn?

Correct Option: A) train_test_split()

Solution and Explanation:
The function train_test_split() from sklearn.model_selection is used to split arrays or matrices into random train and test subsets.




79. What is the purpose of StandardScaler in Scikit-learn?

Correct Option: C) Standardize features by removing the mean and scaling to unit variance

Solution and Explanation:
StandardScaler standardizes a dataset’s features so that they have a mean of 0 and a variance of 1. This is essential for many ML models.




80. How do you import a linear regression model from Scikit-learn?

Correct Option: B) from sklearn.linear_model import LinearRegression

Solution and Explanation:
Scikit-learn's LinearRegression class is located in the linear_model module and is used for creating regression models.
