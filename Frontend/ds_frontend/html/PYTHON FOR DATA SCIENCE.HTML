- p-value ≥ 0.05: Fail to reject the null hypothesis (not significant)

48. What is the purpose of a confusion matrix in classification tasks?

Correct Option: A) To evaluate the performance of a classification model

Solution and Explanation:
A **confusion matrix** summarizes the performance of a classification model. It shows the counts of true positives (TP), true negatives (TN), false positives (FP), and false negatives (FN).

**Key metrics derived from it:**
- **Accuracy:** (TP + TN) / Total
- **Precision:** TP / (TP + FP)
- **Recall:** TP / (TP + FN)
- **F1 Score:** 2 * (Precision * Recall) / (Precision + Recall)

53. What does the term "accuracy" measure in a classification model?

- **TN:** True Negatives
- **FP:** False Positives
- **FN:** False Negatives

55. Which evaluation metric is most suitable for imbalanced classification datasets?

Correct Option: D) F1 Score

Solution and Explanation:
When dealing with **imbalanced datasets**, the **F1 Score** is a better evaluation metric than accuracy. It considers both precision and recall.

**Formula:**
\[ F1 = 2 \times \frac{Precision \times Recall}{Precision + Recall} \] - **Precision:** Correct positive predictions / Total predicted positives
- **Recall:** Correct positive predictions / Total actual positives

57. Which algorithm is commonly used for partitioning data into clusters?

Correct Option: A) K-means Clustering

Solution and Explanation:
**K-means Clustering** is one of the most widely used partitioning algorithms. It divides data into K clusters by minimizing the variance within each cluster.

**Steps in K-means Clustering:**
1. Choose the number of clusters (K).
2. Initialize centroids randomly.
3. Assign each data point to the nearest centroid.
4. Update centroids by calculating the mean of points in each cluster.
5. Repeat steps 3-4 until convergence.

59. Which of the following is a popular algorithm for dimensionality reduction?

Correct Option: A) Principal Component Analysis (PCA)

Solution and Explanation:
**Principal Component Analysis (PCA)** is widely used for dimensionality reduction. It transforms the original data into a new set of variables called principal components, which capture the maximum variance.

**Steps in PCA:**
1. Standardize the data.
2. Compute the covariance matrix.
3. Calculate eigenvalues and eigenvectors.
4. Select the top principal components.
5. Project the data onto these components.




60. Which evaluation metric is commonly used to assess the performance of clustering algorithms?

Correct Option: D) Silhouette Score

Solution and Explanation:
**Silhouette Score** measures how similar an object is to its own cluster compared to other clusters. A higher silhouette score indicates better-defined clusters.

**Formula:**
\[ s = \frac{b - a}{\max(a, b)} \] - **a:** Average distance to other points in the same cluster
- **b:** Average distance to points in the nearest cluster
