Unit 1: Physical Chemistry

1. What is the default port number for SonarQube?

Solution and Explanation:
SonarQube's default port is 8080, commonly used for web applications.




2. Which port is used by Nexus Repository Manager by default?

Solution and Explanation:
Nexus Repository Manager uses port 8081 by default for accessing its web interface.




3. Apache Tomcat listens on which default port for HTTP?

Solution and Explanation:
The default HTTP port for Apache Tomcat is 8080, often used for web servers.




4. What is the default port for the Kubernetes API Server?

Solution and Explanation:
The Kubernetes API Server uses port 6443 for secure communications.




5. Which port does Prometheus use by default?

Solution and Explanation:
Prometheus, a monitoring system, uses port 9090 by default.




6. What is the default port for Elasticsearch in the ELK stack?

Solution and Explanation:
Elasticsearch, a key component of the ELK stack, uses port 9200 by default for HTTP communication. This port is essential for sending search and indexing requests.




7. Which port is commonly used for HTTPS?

Solution and Explanation:
HTTPS (HyperText Transfer Protocol Secure) uses port 443 by default, providing encrypted communication over the web.




8. What is the default port for SSH?

Solution and Explanation:
Secure Shell (SSH) uses port 22 by default, allowing secure remote login to systems.




9. Which port is used by PostgreSQL by default?

Solution and Explanation:
PostgreSQL, a powerful database system, runs on port 5432 by default.




10. Node.js applications commonly use which default port?

Solution and Explanation:
Many Node.js applications run on port 3000 by default, though it can be configured differently.




11. What is the default port for etcd in Kubernetes?

Solution and Explanation:
etcd, the key-value store used in Kubernetes, primarily communicates over port 2379 by default.




12. Which port is used by the Kubernetes scheduler by default?

Solution and Explanation:
The Kubernetes scheduler listens on port 10251 by default for internal communication.




13. What is the default port for the Kubernetes API proxy?

Solution and Explanation:
The Kubernetes API proxy typically listens on port 10256 by default.




14. What is the default service port range for Kubernetes services?

Solution and Explanation:
The default service port range in Kubernetes spans from 30000-32767, used for NodePort services.




15. What is the default port range for Kibana in ELK stack?

Solution and Explanation:
Kibana, the visualization tool in the ELK stack, listens on port 5601 by default.
