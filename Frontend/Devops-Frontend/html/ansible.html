ANSIBLE QUIZ

1. What type of tool is Ansible?

Solution and Explanation:
Ansible is a Configuration Management Tool, used for automating infrastructure management.




2. Which language is used for writing Ansible playbooks?

Solution and Explanation:
Ansible playbooks are written in YAML, a human-readable data serialization language.




3. Which of the following is the core component of Ansible?

Solution and Explanation:
The core component of Ansible is the Ansible Engine, which performs the automation tasks.




4. How does Ansible primarily connect to nodes?

Solution and Explanation:
Ansible connects to nodes primarily via SSH, which is a secure network protocol.




5. What command is used to check the Ansible version?

Solution and Explanation:
To check the Ansible version, you can use the command ansible --version.




6. What is an Ansible playbook?

Solution and Explanation:
An Ansible playbook is a YAML file containing tasks that automate configuration management.




7. What are Ansible modules?

Solution and Explanation:
Ansible modules are executable programs or scripts for automating tasks.




8. What are Ansible facts?

Solution and Explanation:
Ansible facts are system properties or information gathered from managed nodes.




9. Which module is used to manage packages in Debian-based systems?

Solution and Explanation:
The apt module is used to manage packages in Debian-based systems like Ubuntu.




10. What does the 'ansible-playbook' command do?

Solution and Explanation:
The ansible-playbook command is used to run an Ansible playbook, which contains tasks and configurations to be applied to nodes.




11. What is the purpose of the Ansible inventory file?

Solution and Explanation:
The Ansible inventory file is used to list managed nodes that Ansible will manage and interact with.




12. Which of the following is a valid inventory file format for Ansible?

Solution and Explanation:
Ansible supports several file formats for its inventory, including YAML, JSON, and INI formats.




13. What is the role of handlers in Ansible?

Solution and Explanation:
Handlers in Ansible execute tasks only when they are notified by other tasks.




14. Which command is used to view the current Ansible configuration?

Solution and Explanation:
The ansible-config command is used to view and manage the current Ansible configuration.




15. What is the purpose of Ansible Vault?

Solution and Explanation:
Ansible Vault is used to encrypt sensitive data, such as passwords or keys, within playbooks and variables.




16. What is the Ansible Galaxy?

Solution and Explanation:
Ansible Galaxy is a repository of Ansible roles that users can share, reuse, and download.




17. Which Ansible command is used to run ad-hoc commands?

Solution and Explanation:
The ansible command is used for running ad-hoc commands on managed hosts.




18. How can you limit the execution of a playbook to a specific group of hosts?

Solution and Explanation:
You can limit execution to specific hosts by using the --limit flag or by specifying hosts in the playbook.




19. What is a task in an Ansible playbook?

Solution and Explanation:
A task in an Ansible playbook is a unit of work that is executed on the managed hosts.




20. Which command is used to encrypt files with Ansible Vault?

Solution and Explanation:
The ansible-vault encrypt command is used to encrypt files within Ansible Vault.




21. How do you declare variables in an Ansible playbook?

Solution and Explanation:
Variables in an Ansible playbook are declared under the vars section.




22. What is a play in Ansible?

Solution and Explanation:
A play in Ansible is a group of tasks that are executed on a set of hosts.




23. Which Ansible module is used to manage system users?

Solution and Explanation:
The user module is used to manage system users in Ansible.




24. What is the default inventory file location for Ansible?

Solution and Explanation:
The default inventory file location for Ansible is /etc/ansible/hosts.




25. What is the 'gather_facts' directive used for?

Solution and Explanation:
The gather_facts directive is used to collect system information from the hosts.




26. How can you run a playbook with increased verbosity?

Solution and Explanation:
You can run a playbook with increased verbosity by using the --verbose flag.




27. What does the 'ansible-pull' command do?

Solution and Explanation:
The ansible-pull command is used to pull configuration from a remote repository.




28. What is the purpose of 'when' statements in Ansible?

Solution and Explanation:
'When' statements in Ansible are used to execute tasks conditionally.




29. Which command can be used to validate an Ansible playbook syntax?

Solution and Explanation:
You can use the --syntax-check flag to validate the syntax of an Ansible playbook.




30. Which directive is used to include other playbooks in Ansible?

Solution and Explanation:
The include directive is used to include other playbooks in Ansible.




31. Which module is used in the following Ansible playbook to manage the installation of a package?

32. In the following playbook, what does the debug module do?
