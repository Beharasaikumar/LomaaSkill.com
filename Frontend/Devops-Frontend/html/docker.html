Docker-Docker-swarm

1. What is Docker primarily used for?

Solution and Explanation:
Docker is primarily used for Containerization of applications, which helps in packaging and running applications in isolated environments.




2. How do you create a Docker image?

Solution and Explanation:
The correct command to create a Docker image is "docker build -t myimage."




3. Which command is used to list all running Docker containers?

Solution and Explanation:
The correct command to list running containers is "docker ps".




4. What is the purpose of a Dockerfile?

Solution and Explanation:
A Dockerfile is used to define a Docker image by specifying all necessary configurations.




5. What command do you use to run a container in the background?

Solution and Explanation:
The correct command to run a container in the background is "docker run -d".




6. How do you stop a running Docker container?

Solution and Explanation:
The correct command to stop a running Docker container is "docker stop container-id".




7. What is a Docker volume used for?

Solution and Explanation:
A Docker volume is used to persist data between container runs.




8. What is Docker Compose used for?

Solution and Explanation:
Docker Compose is used for defining and running multi-container applications.




9. What is the purpose of the "ENTRYPOINT" instruction in a Dockerfile?

Solution and Explanation:
The "ENTRYPOINT" instruction is used to define the image's default command.




10. How can you scale a service in Docker Compose?

Solution and Explanation:
The correct way to scale a service in Docker Compose is by using the "scale" option.




11. What command is used to remove a Docker container?

Solution and Explanation:
The correct command to remove a Docker container is "docker rm container-id".




12. Which command would you use to see the history of a Docker image?

Solution and Explanation:
The correct command to see the history of a Docker image is "docker history".




13. What does the "-d" flag do in the "docker run" command?

Solution and Explanation:
The "-d" flag is used to run the container in the background.




14. How can you check the Docker version installed on your system?

Solution and Explanation:
Both "docker version" and "docker --version" can be used to check the Docker version.




15. What is the default network mode for a Docker container?

Solution and Explanation:
The default network mode for a Docker container is Bridge.




16. What does the "docker exec" command do?

Solution and Explanation:
The "docker exec" command is used to execute a command in a running container.




17. Given the following Dockerfile CMD instruction:

CMD ["npm", "start"]

You don't want to change the CMD in the Dockerfile, but you want to run the container using 'node server.js'. How can you achieve this?

Solution and Explanation:
The correct way is to use the command: docker run image_name node server.js.




18. Which command is used to build a Docker image from a Dockerfile?

Solution and Explanation:
The correct command is docker build -f Dockerfile.




19. How can you remove unused Docker images?

Solution and Explanation:
The correct command is docker image prune.




20. What is the purpose of Docker's overlay network?

Solution and Explanation:
Docker's overlay network is used to connect containers across different hosts.




21. How can you check the logs of a specific Docker container?

Solution and Explanation:
You can check the logs using "docker logs container-id".




22. What is the purpose of Docker Hub?

Solution and Explanation:
Docker Hub is used to store Docker images.




23. How do you view the details of a specific Docker image?

Solution and Explanation:
You can use "docker inspect" to view the details of a Docker image.




24. What command is used to list all Docker networks?

Solution and Explanation:
The command "docker network list" is used to list all Docker networks.




25. What is the difference between "docker run" and "docker create"?

Solution and Explanation:
The "docker run" command creates and starts a container, while "docker create" only creates it.




26. What is a multi-stage build in Docker?

Solution and Explanation:
A multi-stage build uses multiple "FROM" statements to create a smaller final image.




27. How do you access the Docker command line interface?

Solution and Explanation:
The Docker CLI can be accessed through Docker Desktop and SSH.




28. What command do you use to create a new Docker network?

Solution and Explanation:
The correct command is "docker network create".




29. What is the "docker-compose up" command used for?

Solution and Explanation:
The "docker-compose up" command is used to start services defined in a Docker Compose file.




30. What command is used to pull an image from a Docker registry?

Solution and Explanation:
The "docker pull" command is used to download an image from a Docker registry.




31. How can you update an existing Docker image?

Solution and Explanation:
An existing Docker image can be updated by pulling the latest version from a registry or by rebuilding the image.




32. What is the purpose of the "docker-compose down" command?

Solution and Explanation:
The "docker-compose down" command is used to stop and remove the resources created by "docker-compose up".




33. How do you specify an environment variable in a Docker container?

Solution and Explanation:
You can specify an environment variable using the "-e" flag in "docker run" or by adding it to the Dockerfile using ENV.




34. What is the default storage driver used by Docker?

Solution and Explanation:
The default storage driver used by Docker is Overlay2.




35. Which command allows you to inspect the settings of a Docker container?

Solution and Explanation:
The command used to inspect a Docker container is "docker inspect container-id".




36. What is the function of the "docker network prune" command?

Solution and Explanation:
The "docker network prune" command is used to remove all unused networks.




37. How can you get the IP address of a running container?

Solution and Explanation:
You can get the IP address of a running container using "docker inspect container-id | grep IP" or "docker network inspect bridge".




38. What is the significance of the "EXPOSE" instruction in a Dockerfile?

Solution and Explanation:
The "EXPOSE" instruction in a Dockerfile is used to specify ports that the container listens on at runtime.




39. How do you manage sensitive data in Docker?

Solution and Explanation:
The best way to manage sensitive data in Docker is by using Docker secrets.




40. What does the "docker stats" command do?

Solution and Explanation:
The "docker stats" command is used to show CPU and memory usage of running containers.




41. How can you limit the memory usage of a Docker container?

Solution and Explanation:
You can limit the memory usage of a Docker container using the "-memory" flag in the "docker run" command.




42. What does the "docker-compose build" command do?

Solution and Explanation:
The "docker-compose build" command builds the images defined in the Docker Compose file.




43. What command do you use to rename a Docker container?

Solution and Explanation:
You can rename a Docker container using "docker rename old_name new_name".




44. What is the role of the Docker daemon when a user runs the command: "docker run nginx -name surya1"?

Solution and Explanation:
The Docker daemon checks if the "nginx" image is present locally (in "/var/lib/docker/images"), and if not, it pulls it from Docker Hub, creates a container, and starts it.




45. What command is used to create a Docker service in Swarm mode?

Solution and Explanation:
The command "docker service create service_name" is used to create a Docker service in Swarm mode.




46. How can you list all Docker images on your system?

Solution and Explanation:
You can list all Docker images on your system using the command "docker images".




47. What is a Docker swarm?

Solution and Explanation:
A Docker Swarm is a cluster of Docker engines working together as a single system.




48. What does the "docker attach" command do?

Solution and Explanation:
The "docker attach" command is used to attach a new terminal to a running container.




49. How do you set a health check for a Docker container?

Solution and Explanation:
You can set a health check for a Docker container using the "HEALTHCHECK" instruction in the Dockerfile.




50. What command can you use to export a Docker container's filesystem?

Solution and Explanation:
You can export a Docker container's filesystem using the command "docker export container-id".
