1. Which type of neural network is commonly used for analyzing medical images?

Correct Option: A) CNN

2. Use pooling layers to reduce the dimensionality.
3. Apply fully connected layers for classification or regression.
4. Output the result based on learned features.

3. Which technique is used to identify tumors in radiological images?

Correct Option: B) Segmentation

Solution and Explanation:
Segmentation is a technique used to partition an image into multiple segments or regions. In medical imaging, segmentation helps isolate tumors and other abnormalities from the surrounding tissues.

**Steps in Segmentation**:
1. Convert the image to grayscale or another suitable format.
2. Apply segmentation algorithms (e.g., thresholding, region growing).
3. Detect the regions corresponding to tumors or anomalies.




4. What does the "semantic segmentation" technique help with in medical imaging?

Correct Option: C) Dividing images into meaningful parts

Solution and Explanation:
Semantic segmentation helps divide an image into regions that correspond to different objects or structures, making it easier to analyze medical images.

**Steps in Semantic Segmentation**:
1. Classify each pixel into a category (e.g., tumor, tissue, etc.).
2. Assign labels to different regions of the image.
3. Create a segmented map of the image.
