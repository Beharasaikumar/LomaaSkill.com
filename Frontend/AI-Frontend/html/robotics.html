2. Which algorithm is commonly used for path planning in robotics?

Correct Option: C) A* (A-Star) algorithm

Solution and Explanation:
The **A* algorithm** is widely used for path planning because it finds the shortest path efficiently while avoiding obstacles.

**Steps in A* Algorithm:**
1. Initialize the open and closed lists.
2. Evaluate the cost (g) and heuristic (h) for each node.
3. Select the node with the lowest total cost (f = g + h).
4. Repeat until the goal is reached.
